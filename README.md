# vm-postinstall

A Python 3 script for configuring fresh Ubuntu 18.04 VMs.

## Version

0.2
    
## Requirements

python >= 3

## Installation

Copy vm-postinstall to /usr/local/bin and make executable

## Usage

Run as sudo from your terminal:
$ sudo vm-postinstall
